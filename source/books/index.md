# 说明

本人整理自己藏书时不严格遵循以下原则：

1. 作者人数多于1时，只取第一个；
2. 作者是外国人时，尽量使用本名而非译名，除非本名特别复杂。如，托尔斯泰。
2. 对于外国书籍，应按实际阅读用的语言收录。如，因为我读的是英文原版，所以收录`The Hobbit`而非`霍比特人`。
3. 对于经典的中译本，应在标题后注明译者。如，`不能承受的生命之轻（许均 译）`，或`生命中不能承受之轻（韩少功 译）`。
4. 对于翻译比较垃圾的中译本，应补上原版书名。如，`代码大全（Code Complete）`。
5. 对于流传广泛的简称，可将其并列于标题后。如，`计算机程序的构造和解释（SICP）`。
