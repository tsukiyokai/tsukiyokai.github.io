# 整理说明

藏书的整理优先遵循下列原则：

1. 作者人数为复数时，只取第一个。
2. 作者是外国人时，尽量使用本名而非译名，除非本名特别复杂。如托尔斯泰。
3. 对于外国书籍，按实际阅读语言收录。如果是读的原版书，就收录`The Hobbit`而非`霍比特人`。
4. 对于经典的中译本，应在标题后注明译者。如`生命中不能承受之轻（韩少功 译）`。
5. 对于广泛流传的书籍简称，可将其并列于标题后。如`计算机程序的构造和解释（SICP）`。
