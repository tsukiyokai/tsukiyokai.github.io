## 题目

编写一个程序判断给定的数是否为丑数。
丑数就是只包含质因数2,3,5的正整数。
说明：1是丑数

## 解法

连续消去所有2,3,5倍的因数，判断最后剩下的数是否为1。

**代码**

```c++
class Solution {
public:
    bool isUgly(int n) {
        if (n < 1) return false;
        while (n % 2 == 0) n /= 2;
        while (n % 3 == 0) n /= 3;
        while (n % 5 == 0) n /= 5;
        return n == 1;
    }
};
```
