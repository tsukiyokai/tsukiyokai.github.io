## 题目

给定一个字符串，请你找出其中不含有重复字符的最长子串的长度。

## 解法1：暴力法

时间复杂度：O(N^3)

## 解法2：滑动窗口算法

时间复杂度：O(N^2)

关键字1：重复字符，即最多只出现一次。
模式识别1：一旦涉及出现次数，需要用到散列表。
关键字2：最长子串
模式识别：涉及子串，考虑滑动窗口。

**代码**

```c++
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        if (s.size() == 0) return 0;
        unordered_set<char> set;
        int maxlen = 0;
        int left = 0;
        for (int right = 0; right < s.size(); right++) {
            while (set.find(s[right]) != set.end())
                set.erase(s[left++]);
            maxlen = max(maxlen, right - left + 1);
            set.insert(s[right]);
        }
        return maxlen;
    }
};
```

**注释**

set实时存放当前窗口内出现的字符
对right的遍历是最主要的遍历，right或left只能前进，不能后退
不断从左缩小窗口直到窗口中不存在与右边界重复的字符
