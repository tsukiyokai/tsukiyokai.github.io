## 题目

给你一个整数x，如果x是一个回文整数，返回true；否则，返回false 。
回文数是指正序和倒序读都是一样的整数。例如，121是回文，而123不是。

## 解法：字符串

**代码**

```c++
class Solution {
public:
    bool isPalindrome(int x) {
        string str = to_string(x);
        for (int i = 0; i < str.size() / 2; i++)
            if (str[i] != str[str.size() - 1 - i]) return false;
        return true;
    }
};
```
