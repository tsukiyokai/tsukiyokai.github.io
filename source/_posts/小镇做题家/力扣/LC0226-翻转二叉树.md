## 题目

镜像翻转一棵二叉树。

## 解法1：递归

**代码**

```c++
class Solution {
public:
    TreeNode* invertTree(TreeNode* root) {
        if (root == nullptr || root->left == nullptr && root->right == nullptr) return root;
        swap(root->left, root->right);
        if (root->left) invertTree(root->left);
        if (root->right) invertTree(root->right);
        return root;
    }
};
```
