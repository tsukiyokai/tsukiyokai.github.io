把m个同样的苹果放在n个同样的盘子里，允许有的盘子空着不放，问共有多少种不同的分法？（用K表示）5，1，1和1，5，1 是同一种分法。

数据范围：0<=m<=10，1<=n<=10。
本题含有多组样例输入。

输入描述: 输入两个int整数
输出描述: 输出结果，int型

输入
7 3
输出
8

```c++
#include <iostream>
#include <string>

using namespace std;

int fun(int m, int n) {
    if (m == 0 || m == 1 || n == 1) return 1;
    if (m < 0) return 0;
    return fun(m, n - 1) + fun(m - n, n);
}

int main() {
    int m, n;
    while (cin >> m >> n) {
        cout << fun(m, n) << endl;
    }

    return 0;
}
```

分析
写状态转移方程：有盘子（至少一个）为空 和 无盘子为空
找递归边界
