## 题目

将一个字符中所有的整数前后加上符号“*”，其他字符保持不变。连续的数字视为一个整数。

注意：本题有多组样例输入。
输入描述:
输入一个字符串
输出描述:
字符中所有出现的数字前后加上符号“*”，其他字符保持不变

示例1
输入
`Jkdi234klowe90a3`
`5151`
输出
`Jkdi*234*klowe*90*a*3*`
`*5151*`

## 解法

```c++
#include<iostream>
#include<string>
using namespace std;

int main() {
    string s;
    while (cin >> s) {
        string res;
        for (size_t i = 0; i < s.length(); i++) {
            if (!isdigit(s[i])) {
                if (i == 0 || !isdigit(s[i - 1]))
                    res += s[i];
                else {
                    res += '*';
                    res += s[i];
                }
            }
            else {
                if (i == 0 || !isdigit(s[i - 1])) {
                    res += '*';
                    res += s[i];
                }
                else {
                    res += s[i];
                }
            }
        }
        if (isdigit(res.back())) res.push_back('*');
        cout << res << endl;
    }
    return 0;
}
```
