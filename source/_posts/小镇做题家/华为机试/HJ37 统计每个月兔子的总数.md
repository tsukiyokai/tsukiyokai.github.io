## 题目

有一只兔子，从出生后第 3 个月起每个月都生一只兔子，小兔子长到第三个月后每个月又生一只兔子，假如兔子都不死，问每个月的兔子总数为多少？
本题有多组数据。

## 解法：DP

这道题与其说是找规律，不如说就是 DP 的思想。

**代码**

```c++
#include <iostream>
using namespace std;

int main() {
    int m;
    while (cin >> m) {
        int a = 1, b = 0, c = 0; // a:一个月兔子数，b：两个月兔子数，c：三个月兔子个数
        while (--m) { // 每过一个月兔子数变化
            c += b;
            b = a;
            a = c;
        }
        cout << a + b + c << endl;
    }
}
```
