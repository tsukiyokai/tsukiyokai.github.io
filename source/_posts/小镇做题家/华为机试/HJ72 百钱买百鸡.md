## 题目

公元前五世纪，我国古代数学家张丘建在《算经》一书中提出了“百鸡问题”：
鸡翁一值钱五，鸡母一值钱三，鸡雏三值钱一。百钱买百鸡，问鸡翁、鸡母、鸡雏各几何？

## 解法：暴力

```c++
#include <iostream>
using namespace std;

int main() {
    int x;
    while (cin >> x) {
        int i, j;
        for (i = 0; i <= 20; i++) {
            for (j = 0; j <= 34; j++) {
                if ((100 - i - j) % 3 == 0 && (5 * i + 3 * j + (100 - i - j) / 3 == 100))
                    cout << i << " " << j << " " << 100 - i - j << endl;
            }
        }
    }

    return 0;
}
```
