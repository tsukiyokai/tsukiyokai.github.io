实现删除字符串中出现次数最少的字符，若多个字符出现次数一样，则都删除。输出删除这些单词后的字符串，字符串中其它字符保持原来的顺序。
注意每个输入文件有多组输入，即多个字符串用回车隔开

输入描述: 字符串只包含小写英文字母, 不考虑非法输入，输入的字符串长度小于等于20个字节。
输出描述: 删除字符串中出现次数最少的字符后的字符串。

```tex
输入
abcdd
aabcddd
输出
dd
aaddd
```

```c++
#include <iostream>
#include <algorithm>
#include <vector>
#include <unordered_map>

using namespace std;

int lesstime(unordered_map<char, int> m) {
    int less = 20;
    for (auto& p : m)
        less = min(less, p.second);
    return less;
}

int main() {
    string s;
    while (cin >> s) {
        unordered_map<char, int> m;
        for (char& c : s)
            m[c] += 1;
        for (char& c : s)
            if (m[c] != lesstime(m))
                cout << c;
        cout << endl;
    }
    return 0;
}
```

两个总结
1 不需要等全部处理妥当再一起输出，可以每碰见一个合适的就输出它，可以少一些麻烦
2 怎么取map里的value最小值？先设一个最小值然后不断更新它，注意，千万不要耿直地把最小值直接设成有可能地最小值。我一开始就是把less设成了0，结果全都不通过，这时我就应该发现了，结果我转手把less又改成了1，试了一下还通过了，结果拿上去运行，通过9/10。记住一定要把人为设定地最小值写成理论最大值，这里可以写成INI_MAX，也可以写成20，因为字符串长度最多也就20，所以出现次数不会多于20。
