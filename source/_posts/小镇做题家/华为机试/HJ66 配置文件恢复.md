## 题目

有6条配置命令，它们执行的结果分别是：

|命令|执行|
|-|-|
|reset|reset what|
|reset board|board fault|
|board add|where to add|
|board delete|no board at all|
|reboot backplane|impossible|
|backplane abort|install first|
|he he|unknown command|

注意：he he不是命令。

为了简化输入，方便用户，以“最短唯一匹配原则”匹配：
1、若只输入一字串，则只匹配一个关键字的命令行。例如输入：r，根据该规则，匹配命令reset，执行结果为：reset what；输入：res，根据该规则，匹配命令reset，执行结果为：reset what；
2、若只输入一字串，但本条命令有两个关键字，则匹配失败。例如输入：reb，可以找到命令reboot backpalne，但是该命令有两个关键词，所有匹配失败，执行结果为：unknown command
3、若输入两字串，则先匹配第一关键字，如果有匹配但不唯一，继续匹配第二关键字，如果仍不唯一，匹配失败。例如输入：r b，找到匹配命令reset board 和 reboot backplane，执行结果为：unknown command。
4、若输入两字串，则先匹配第一关键字，如果有匹配但不唯一，继续匹配第二关键字，如果唯一，匹配成功。例如输入：b a，无法确定是命令board add还是backplane abort，匹配失败。
5、若输入两字串，第一关键字匹配成功，则匹配第二关键字，若无匹配，失败。例如输入：bo a，确定是命令board add，匹配成功。
6、若匹配失败，打印“unknown command”

输入：
多行字符串，每行字符串一条命令
输出：
执行结果，每条命令输出一行

## 解法：阅读理解，外加一些小技巧

```c++
#include <iostream>
#include <string>
#include <sstream>
using namespace std;

bool match(string str, string s) {
    return str.find(s) != string::npos;
}

int main() {
    string cmd[6] = {
        "reset",
        "reset board",
        "board add",
        "board delete",
        "reboot backplane",
        "backplane abort" };
    string exe[7] = {
        "reset what",
        "board fault",
        "where to add",
        "no board at all",
        "impossible",
        "install first",
        "unknown command" };
    string str;
    while (getline(cin, str)) {
        string s1, s2, tmp1, tmp2;
        stringstream ss(str);
        ss >> s1 >> s2;
        int exei = 6;
        if (s2.empty()) // 若只输入一字串，则只匹配一个关键字的命令行。
            match(cmd[0], s1) ? exei = 0 : exei = 6;
        else { // 若输入两字串
            bool flag = false;
            for (int i = 1; i < 6; i++) {
                stringstream bincmd(cmd[i]);
                bincmd >> tmp1 >> tmp2;
                if (match(tmp1, s1) && match(tmp2, s2)) {
                    if (flag == 0) {
                        flag = true;
                        exei = i;
                    }
                    else {
                        exei = 6;
                        break;
                    }
                }
            }
        }
        cout << exe[exei] << endl;
    }
    return 0;
}
```

知识点1：`string::npos`表示没搜索到的情况。
知识点2：`stringstream`用来再处理已有的字符串，比如最自然的用法是让它以空格为界自动分成几个小字符串。
知识点3：`bool flag`用来描述状态是否改变过或现在是否满足要求。
