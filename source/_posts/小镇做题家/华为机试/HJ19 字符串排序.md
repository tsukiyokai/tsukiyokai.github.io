给定 n 个字符串，请对 n 个字符串按照字典序排列。

## 解法

```c++
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    int n;
    vector<string> v;
    cin >> n;
    while (n--) {
        string str;
        cin >> str;
        v.push_back(str);
    }
    sort(v.begin(), v.end(), cmp);
    for (auto word : v)
        cout << word << endl;

    return 0;
}
```

一个坑是 while (n--)里面的 cin >> str，我本来写的是 getline 结果发现总是输入输出有问题。我突然想起来以前的笔记，cin>>和 getline 混用时的大坑。但是这里比那个写的还要复杂一点，因为不是严格混用。最后我还是改成 cin>>str 了，有时间再慢慢研究 getline 到底怎么用
