题目描述
找出字符串中第一个只出现一次的字符

输入描述:
输入几个非空字符串

输出描述:
输出第一个只出现一次的字符，如果不存在输出-1

示例1
输入
asdfasdfo
aabb
输出
o
-1

```c++
#include <iostream>
#include <unordered_map>

using namespace std;

int main() {
    string s;
    while (cin >> s) {
        unordered_map<char, int> m;
        for (char& c : s)
            m[c] += 1;
        char res = ' ';
        for (char& c : s)
            if (m[c] == 1) {
                res = c;
                break;
            }
        if (res == ' ') cout << "-1" << endl;
        else cout << res << endl;
    }
    return 0;
}
```
