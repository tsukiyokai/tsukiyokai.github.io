题目描述
蛇形矩阵是由1开始的自然数依次排列成的一个矩阵上三角形。

例如，当输入5时，应该输出的三角形为：

1 3 6 10 15
2 5 9 14
4 8 13
7 12
11

请注意本题含有多组样例输入。

输入描述:
输入正整数N（N不大于100）

输出描述:
输出一个N行的蛇形矩阵。

示例1
输入
4
输出
1 3 6 10
2 5 9
4 8
7

```c++
#include <iostream>

using namespace std;

int main() {
    int n;
    while (cin >> n) {
        int beg = 1;
        for (int i = 1; i <= n; i++) {
            cout << beg;
            int tmp = beg;
            for (int j = 2; j <= n + 1 - i; j++) {
                tmp += j + (i - 1);
                cout << ' ' << tmp;
            }
            cout << '\n';
            beg += i;
        }
    }

    return 0;
}
```

beg表示每行的开头元素
