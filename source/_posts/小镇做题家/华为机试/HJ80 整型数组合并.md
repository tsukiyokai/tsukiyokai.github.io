题目：将两个整型数组按照升序合并，并且过滤掉重复数组元素。输出时相邻两数之间没有空格。

```c++
#include <iostream>
#include <vector>
#include <string>
#include <sstream>
#include <set>
using namespace std;

int main() {
    int N, M;

    while (cin >> N) {
        set<int> uset;
        while (N--) {
            int tmp; cin >> tmp;
            uset.insert(tmp);
        }
        cin >> M;
        while (M--) {
            int tmp; cin >> tmp;
            uset.insert(tmp);
        }
        for (auto& num : uset) cout << num;
        cout << endl;
    }

    return 0;
}
```

一大组两小组数据的输入方法：对第一小组进行循环的判断，循环中单独输入第二小组。
