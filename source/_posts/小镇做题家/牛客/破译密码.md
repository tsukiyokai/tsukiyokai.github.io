## 题目

译码规则：

1. 字母如A开头，串中所有字母和数字顺延1位，其他不变。
2. 数字如1开头，串中所有字母和数字顺延1位，其他不变。
3. 符号如_开头，视为字母A开头。
4. Z顺延一位变回A，9顺延一位变回0

## 代码

```c++
#include<iostream>
#include<string>
using namespace std;

string trans(string before) {
    string after = before;
    int diff;
    if (before[0] >= 'A' && before[0] <= 'Z') diff = before[0] - 'A' + 1;
    else if (before[0] >= 'A' && before[0] <= 'Z') diff = before[0] - '0';
    else diff = 1;
    for (auto& c : after) {
        if (c >= 'A' && c <= 'Z') c = static_cast<char>('A' + (c - 'A' + diff) % 26);
        if (c >= '0' && c <= '9') c = static_cast<char>('0' + (c - '0' + diff) % 10);
    }
    return after;
}

int main() {
    string str;
    while (getline(cin, str)) {
        cout << trans(str) << endl;
    }

    return 0;
}
```
