## 题目

给你一个字符串 s，由若干单词组成，单词之间用空格隔开。返回字符串中最后一个单词的长度。如果不存在最后一个单词，请返回 0。
单词是指仅由字母组成、不包含任何空格字符的最大子字符串。

## 解法

**代码**

```c++
class Solution {
public:
    int lengthOfLastWord(string s) {
        if (allspace(s)) return 0;
        s = trim(s);
        auto ed = s.end() - 1;
        while (*ed != ' ' && ed != s.begin()) {
            --ed;
        }
        if (ed == s.begin()) return s.size();
        return s.end() - ed - 1;
    }

    bool allspace(string str) {
        for (char it : str) {
            if (it != ' ') return false;
        }
        return true;
    }

    string& trim(string& str) {
        if (str.empty()) return str;
        str.erase(0, str.find_first_not_of(" "));
        str.erase(str.find_last_not_of(" ") + 1, str.length());
        return str;
    }
};
```

**注释**

本来只写了 5 行代码就通过了常规的测试用例。结果提交的时候在各种畸形用例上翻车。翻车的两个用例："a " 和 " a "，连续翻车了 14 次，第 15 次才完全通过。代码也在不知不觉间扩充到接近 30 行了。想一次性考虑到各种极端情况还是挺困难的，应多注意锻炼这方面的能力。Think twice, code once. 并不是那么容易做到。

erase()用法：从 vector 中删除一个元素（position），或者范围元素（[first, last)）。返回一个 iterator，指向删除元素的下一个元素。所以在用 for 循环删除元素时，迭代器不用++指向下一个元素，erase()执行后，自动返回一个迭代器指向下一个元素。
