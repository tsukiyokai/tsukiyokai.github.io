## 题目

给定一个正整数num，编写一个函数，如果num是一个完全平方数，则返回True，否则返回False。
说明：不要使用任何内置的库函数，如sqrt。

## 解法1：牛顿迭代法

**代码**

```c++
class Solution {
public:
    bool isPerfectSquare(int num) {
        if (mySqrt(num) * mySqrt(num) == num) return true;
        return false;
    }

    int mySqrt(int x) {
        if (x == 0) return 0;
        double C = x, x0 = x;
        while (true) {
            double xi = 0.5 * (x0 + C / x0);
            if (fabs(x0 - xi) < 1e-7) break;
            x0 = xi;
        }
        return (int)x0;
    }
};
```

## 解法2：找规律

利用 1+3+5+7+9+…+(2n-1)=n^2，即完全平方数肯定是前n个连续奇数的和

**代码**

```c++
class Solution {
public:
    bool isPerfectSquare(int num) {
        int num1 = 1;
        while (num > 0) {
            num -= num1;
            num1 += 2;
        }
        return num == 0;
    }
};
```

**分析**

1=1
4=1+3
9=1+3+5
16=1+3+5+7
...
以此类推，模仿它可以使用一个while循环，不断减去一个从1开始不断增大的奇数，若最终能减到0，说明是完全平方数，否则不是。

原理：(n+1)^2-n^2=2n+1，即相邻两个完全平方数的差必为奇数，反过来也成立，即任一奇数必定是相邻两个完全平方数的差。
