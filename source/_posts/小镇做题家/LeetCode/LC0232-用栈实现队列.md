## 题目

请你仅使用两个栈实现先入先出队列。
队列应当支持一般队列的支持的所有操作(push、pop、peek、empty)。

## 解法

By making pop operation costly

**代码**

```c++
class MyQueue {
private:
    stack<int> s1;
    stack<int> s2;
    int front;
public:
    MyQueue() { }

    void push(int x) {
        if (s1.empty()) front = x;
        s1.push(x);
    }

    int pop() {
        if (s1.empty() && s2.empty()) {
            exit(0);
        }
        if (s2.empty()) {
            while (!s1.empty()) {
                s2.push(s1.top());
                s1.pop();
            }
        }
        int x = s2.top();
        s2.pop();
        return x;
    }

    int peek() {
        if (!s2.empty()) return s2.top();
        return front;
    }

    bool empty() {
        return s1.empty() && s2.empty();
    }
};
```

**注释**

额外维护一个整型变量 front，用以保存队头元素。
