## 题目

给你两个有序整数数组 nums1 和 nums2，请你将 nums2 合并到 nums1 中，使 nums1 成为一个有序数组。初始化 nums1 和 nums2 的元素数量分别为 m 和 n。你可以假设 nums1 的空间大小等于 m+n，这样它就有足够的空间保存来自 nums2 的元素。

输入：nums1=[1,2,3,0,0,0], m=3, nums2=[2,5,6], n=3
输出：[1,2,2,3,5,6]

## 解法

**代码**

```c++
class Solution {
public:
    void merge(vector<int>& nums1, int m, vector<int>& nums2, int n) {
        trim(nums1, m);
        trim(nums2, n);
        while (nums2.size() != 0) {
            nums1.push_back(*nums2.begin());
            nums2.erase(nums2.begin());
        }
        sort(nums1.begin(), nums1.end());
    }

    void trim(vector<int>& nums, int m) {
        if (nums.size() > m) nums.erase(nums.end() - (nums.size() - m), nums.end());
    }
};
```

**注释**

trim()用来修理掉所给数组右边的占位 0
