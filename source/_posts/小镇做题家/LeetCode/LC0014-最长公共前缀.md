## 题目

编写一个函数来查找给定一组字符串的最长公共前缀。如果不存在公共前缀，返回空字符串`""`。

## 解法1：逐词匹配

我们从一个例子开始。假设有两个字符串`geeksforgeks`和`geeks`。它们中最长的共同前缀是什么？是`geeks`。现在让我们介绍另一个词`geek`。那么这三个词中最长的共同前缀是什么呢？是`geek`。

我们可以看到最长的公共前缀具有关联属性，即：
`LCP(string1,string2,string3)=LCP(LCP(string1,string2),string3)`

因此，我们可以利用上述结合性质来寻找给定字符串的LCP。我们逐一计算每个给定字符串的LCP与目前为止的LCP。最终结果将是所有字符串中最长的公共前缀。

![](https://media.geeksforgeeks.org/wp-content/cdn-uploads/longest_common_prefix-1024x485.png)

**代码**

```c++
class Solution {
public:
    string longestCommonPrefix(vector<string>& strs) {
        if (strs.size() == 0) return "";
        string prefix = strs[0];
        for (int i = 1; i < strs.size(); i++) {
            prefix = commonPrefixUtil(prefix, strs[i]);
        }
        return prefix;
    }

    string commonPrefixUtil(string str1, string str2) {
        string result;
        int len1 = str1.length(), len2 = str2.length();
        for (int i = 0, j = 0; i <= len1 - 1 && j <= len2 - 1; i++, j++) {
            if (str1[i] != str2[j]) break;
            result.push_back(str1[i]);
        }
        return result;
    }
};
```

**分析**

时间复杂度：O(n×m)。n为字符串数量，m为最长字符串的长度。
空间复杂度：O(m)

## 解法2：逐字匹配

![](https://media.geeksforgeeks.org/wp-content/cdn-uploads/longest_common_prefix2.jpg)

![](https://media.geeksforgeeks.org/wp-content/cdn-uploads/longest_common_prefix5.jpg)

**代码**

```c++
class Solution {
public:
    string longestCommonPrefix(vector<string>& strs) {
        if (strs.size() == 0) return "";
        int minlen = findMinLength(strs, strs.size());
        string prefix;
        char curr = 0;
        for (int i = 0; i < minlen; i++) {
            curr = strs[0][i];
            for (int j = 1; j < strs.size(); j++) {
                if (strs[j][i] != curr) return prefix;
            }
            prefix.push_back(curr);
        }
        return prefix;
    }

    int findMinLength(vector<string>& strs, int n) {
        int min = strs[0].length();
        for (int i = 1; i < n; i++)
            if (strs[i].length() < min) min = strs[i].length();
        return min;
    }
};
```

## 解法3：排序

对字符串数组进行排序，并找到排序数组中第一个和最后一个字符串的公共前缀。

**代码**

```c++
class Solution {
public:
    string longestCommonPrefix(vector<string>& strs) {
        int n = strs.size();
        if (n == 0) return "";
        if (n == 1) return strs[0];
        sort(strs.begin(), strs.end());
        int minlen = min(strs[0].size(), strs[n - 1].size());
        string first = strs[0], last = strs[n - 1];
        int i = 0;
        while (i < minlen && first[i] == last[i]) i++;
        string prefix = first.substr(0, i);
        return prefix;
    }
};
```
