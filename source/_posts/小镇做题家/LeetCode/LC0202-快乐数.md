## 题目

编写一个算法来判断一个数 n 是不是快乐数。

快乐数定义为：
对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和。
然后重复这个过程直到这个数变为 1，也可能是无限循环但始终变不到 1。
如果可以变为 1，那么这个数就是快乐数。
如果 n 是快乐数就返回 true；不是则返回 false 。

## 解法：双指针

此题可以拆分成两个模块，一是求各位平方之和，二是循环和跳出循环。
难点在于何时跳出循环，显然是自己正处于一个无限循环中的时候。那如何知道自己是否正在处于一个循环中呢？想到了另一道题环形链表中的弗洛伊德判圈算法。于是采用快慢指针的方法。

需要清楚一点：如果自身永远不能到达 1，说明自己必定处在一个无限循环中，注意是无限循环而非无限不循环。有没有可能无限不循环呢（像无理数那样）？答案是不会，因为 10 以内的整数个数是有限的。

区分：
无限循环但始终变不到 1
无限不循环但始终变不到 1

```c++
class Solution {
public:
    int bitSquareSum(int n) {
        int sum = 0;
        while (n) {
            int bit = n % 10;
            sum += bit * bit;
            n = n / 10;
        }
        return sum;
    }

    bool isHappy(int n) {
        int slow = n, fast = n;
        do {
            slow = bitSquareSum(slow);
            fast = bitSquareSum(fast);
            fast = bitSquareSum(fast);
        } while (slow != fast);
        return slow == 1;
    }
};
```
