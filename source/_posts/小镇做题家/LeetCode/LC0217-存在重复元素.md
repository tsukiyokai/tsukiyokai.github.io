## 题目

给定一个整数数组，判断是否存在重复元素。
如果存在一值在数组中出现至少两次，函数返回 true。如果数组中每个元素都不相同，则返回 false 。

输入: [1,2,3,1]
输出: true

输入：[]
返回：false

## 解法 1：暴力法

两层 for

## 解法 2：排序

先排序，在遍历，只遍历一遍

**代码**

```c++
class Solution {
public:
    bool containsDuplicate(vector<int>& nums) {
        if (nums.size() == 0) return false;
        sort(nums.begin(), nums.end());
        for (int i = 0; i < nums.size() - 1; ++i) {
            if (nums[i] == nums[i + 1]) return true;
        }
        return false;
    }
};
```
