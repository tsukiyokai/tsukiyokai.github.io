## 题目

给你一个字符串 s，找到 s 中最长的回文子串。

## 解法1：暴力（超时）

暴力不行，虽然在NC可以通过，但是NC的用例简单，而且时间没啥限制（1000ms），而LC就比较严格，在LC做题不要想着用暴力。

```c++
class Solution {
public:

    bool isPalindrome(string s) {
        for (int i = 0; i < s.size() / 2; i++)
            if (s[i] != s[s.size() - 1 - i]) return false;
        return true;
    }

    string longestPalindrome(string s) {
        ios_base::sync_with_stdio(false);
        cin.tie(0);

        string tmp;
        for (int len = s.length(); len > 0; len--) {
            for (int it = 0; it <= s.length() - len; it++) {
                if (isPalindrome(s.substr(it, len))) {
                    tmp = s.substr(it, len);
                    break;
                }
                continue;
            }
            if (!tmp.empty()) break;
            else continue;
        }
        return tmp;
    }
};
```
