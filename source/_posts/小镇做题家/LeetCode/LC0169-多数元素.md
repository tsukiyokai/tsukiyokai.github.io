## 题目

给定一个大小为n的数组，找到其中的多数元素。多数元素是指在数组中出现次数大于n/2的元素。你可以假设数组是非空的，并且给定的数组总是存在多数元素。

## 解法1：排序

**代码**

```c++
class Solution {
public:
    int majorityElement(vector<int>& nums) {
        sort(nums.begin(), nums.end());
        return nums[nums.size() / 2];
    }
};
```

**注释**

用std::sort()给数组排序，升序降序无所谓。正中间元素的右邻域那个元素一定是所谓的多数元素。注意一点，size()的范围是从1到n，如果数组非空的话。而nums[]索引的范围是0~n-1。

## 解法2：摩尔投票法

**代码**

```c++
class Solution {
public:
    int majorityElement(vector<int>& nums) {
        int count = 0;
        int res = nums[0];
        for (int i = 0; i < nums.size(); ++i) {
            if (res == nums[i]) count++;
            else count--;
            if (count == 0) res = nums[i + 1];
        }
        return res;
    }
};
```

**注释**

假设首元素是多数元素，从首元素开始遍历，遇到相同元素count+1，遇到不同元素count-1。当count减到0时，更新存储多数元素的变量为下一个元素。遍历结束时，最后留下的元素就是多数元素。
