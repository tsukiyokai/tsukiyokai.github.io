## 题目

请你仅使用队列实现一个后入先出(LIFO)的栈，并支持普通队列的全部四种操作(push、top、pop 和 empty)。

## 解法

只使用一个队列
By making push operation costly

**代码**

```c++
class MyStack {
    queue<int> nums;
public:
    MyStack() {}

    void push(int x) {
        nums.push(x);
        for (int i = 0; i < nums.size() - 1; i++) {
            nums.push(nums.front());
            nums.pop();
        }
    }

    int pop() {
        int num = nums.front(); nums.pop();
        return num;
    }

    int top() {
        return nums.front();
    }

    bool empty() {
        return nums.empty();
    }
};
```

**注释**

把新插入队列的结点前面的所有结点都重新插入到性新结点后面。
