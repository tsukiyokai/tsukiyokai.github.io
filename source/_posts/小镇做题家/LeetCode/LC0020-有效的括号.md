## 题目

给定一个只包括'('，')'，'{'，'}'，'['，']'的字符串 s，判断字符串是否有效。

有效字符串需满足：
左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。

## 解法：栈

**代码**

```c++
class Solution {
public:
    bool isValid(string s) {
        stack<char> st;
        char tmp = 0;
        for (auto c : s) {
            if (st.empty() || c == '(' || c == '[' || c == '{') {
                st.push(c);
                continue;
            }
            if (c == ')') {
                tmp = st.top();
                st.pop();
                if (tmp == '(') continue;
                return false;
            }
            if (c == ']') {
                tmp = st.top();
                st.pop();
                if (tmp == '[') continue;
                return false;
            }
            if (c == '}') {
                tmp = st.top();
                st.pop();
                if (tmp == '{') continue;
                return false;
            }
        }
        if (st.empty()) return true;
        return false;
    }
};
```
