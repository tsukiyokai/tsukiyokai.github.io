## 题目

给你一个整数 n，请你帮忙计算并返回该整数各位数字之积与各位数字之和的差。

## 解法：字符串

用到的库函数：
to_string(): Convert numerical value to string

**代码**

```c++
class Solution {
public:
    int subtractProductAndSum(int n) {
        int mul = 1, sum = 0;
        string num = to_string(n);
        for (auto i : num) {
            sum += i - '0';
            mul *= i - '0';
        }
        return mul - sum;
    }
};
```

**注释**

把整数转换成字符串字面值，然后对于字符串，使用基于范围的 for 循环，进行计算。
