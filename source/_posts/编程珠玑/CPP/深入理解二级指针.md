## 二级指针概念

一级指针存储变量的地址，二级指针存储一级指针的地址。当我们想通过函数改变某个变量的值时，我们向函数传递指向该变量的指针；而如果我们想改变一个指针的值，例如改变指针的指向，此时我们传递指向该指针的指针，即二级指针。

总体上应把握住这样一个原则：将指针作为函数参数时，看是为了修改指针指向的值，还是为了修改指针本身。前者传入一级指针，后者传入二级指针。因为指针本身也是一种变量，它也有相应的地址，对指针取地址就得到了指针这个对象的地址。下面以具体代码来解释各级指针的含义。

```c
#include <stdio.h>

int main() {
    int a = 10;
    int* p = &a;
    int** q = &p;
    printf("a   = %d\n", a);
    printf("&a  = %p\n", &a);
    printf("p   = %p\n", p);
    printf("&p  = %p\n", &p);
    printf("*p  = %d\n", *p);
    printf("q   = %p\n", q);
    printf("&q  = %p\n", &q);
    printf("*q  = %p\n", *q);
    printf("**q = %d\n", **q);
    return 0;
}
```

| 用法          | 含义                                               |
| ------------- | -------------------------------------------------- |
| int a= 10     | a 的类型为 int，值为 10                            |
| int\* p= &a   | p 的类型为指向 int 的指针，值为 a 的地址           |
| int\*\* q= &p | q 的类型为指向指向 int 的指针的指针，值为 p 的地址 |
| a             | a 的内容                                           |
| &a            | a 的地址                                           |
| p             | p 的内容，即 a 的地址                              |
| &p            | p 的地址                                           |
| \*p           | p 解引用，相当于 a 先取地址再解引用，即 a          |
| q             | q 的内容，即 p 的地址                              |
| &q            | q 的地址                                           |
| \*q           | q 解引用，即 p 先取地址再解引用，即 p，即 a 的地址 |
| \*\*q         | q 解引用两次，即 a 先取地址再解引用，即 a          |

可见，二级指针也是指针，用指针的思想来理解这个问题就好，区别只是在于一级指针是由于存放普通变量的地址的，二级指针是由于存放指针变量的地址的。另外需要注意的就是 C 语言在进行函数调用时，实参的传递采用的是实参值的一份拷贝。如果要在函数内部改变变量的值，就要传入指针，如果要在函数内部改变指针的值，就需要传入二级指针。

## 二级指针应用

首先，二级指针作为函数参数。在函数外部定义一个指针 p，在函数内给指针赋值，函数结束后对指针 p 生效，此时需要二级指针。其次，二级指针可用于各种链表操作。链表操作可能会改变头指针的实际指向。简单来说，修改头指针则必须传递头指针的地址，若只修改指针指向的对象，则传递头指针即可。这与普通变量类似，当需要修改普通变量的值，需传递该普通变量的地址，否则传递这个变量的拷贝。使用二级指针，很方便就修改了传入的结点一级指针的值。如果用一级指针，则只能通过指针修改指针所指内容，却无法修改指针的值，也就是指针所指的内存块。所以创建链表和销毁链表需要二级指针或者一级指针引用。
