## 用位运算代替一些普通操作

左移就是乘以2
右移就是除以2

```c
n / 2; 等价于 n >> 1
n / 4; 等价于 n >> 2
n / 8; 等价于 n >> 3
```

## 用位运算判断奇偶性

与1，等价于与0...0001
任何数与0都是0，任何数与1相与都不变
总而言之，与1就是取二进制的末一尾
（进一步拓展，与0xF就是取二进制的后四位）

```c
bool IsOdd(int i) { return i & 1; }
```

## n&(n-1)的妙用

`n&(n-1)`的功能：消去n的二进制表示中的最右边的1

判断一个正整数n是否为2的幂次方。如果一个数是2的幂次方，意味着n的二进制表示中，只有一个位是1，其他都是0。例如：
2^0=0……0001
2^1=0……0010
2^2=0……0100
所以，对一个正整数n执行`n&(n-1)`，得到0，说明是2的幂次方，否则不是。例如：

```c
bool judege(int n) { return (n & (n - 1)) == 0; }
```
   
判断正整数n的二进制表示中有多少个1。每执行一次n&(n-1)时，n的二进制表示中消去一位最右边的1，消去所有1时，n变成0。
欲知n中有多少个1，只需计算总共执行了多少次。比如：

```c
while (n) {
	count++;
	n = (n - 1) & n;
}
return count;
```

计算整数m和n有多少位二进制位不同，首先执行`m^n`，相同位取得0，相异位取得1。现在转化成第2类问题，检查上一步结果中有多少位1即可。

## XOR的妙用

性质1：两个相同的数异或，运算结果为0，例如`n^n=0`
性质2：任何数和0异或，运算结果不变，例如`n^0=n`
性质3：支持交换律和结合律，例如`x^(y^x)=(x^y)^x`

例题：
数组中，只有一个数出现一次，剩下都出现偶数次，请找出仅出现一次的数。

常规操作：
一边遍历数组一边用哈希表统计元素出现的次数，最后再遍历哈希表，看看哪个数只出现了一次。时间复杂度为O(n)，空间复杂度为O(n)。

特殊技巧：
利用异或的几个性质，两个相同的数异或的结果是0，一个数和0异或的结果是它本身，所以我们把这组整型全部异或一遍。
例如这组数据是`1,2,3,4,5,1,2,3,4`。其中5只出现了一次，其他都出现了两次或偶数次，把他们全部异或，根据异或的交换律和结合律，可得`1^2^3^4^5^1^2^3^4=(1^1)^(2^2)^(3^3)^(4^4)^5=0^0^0^0^5=5`。

通过这种方法，可以把空间复杂度降低到O(1)，而时间复杂度不变。例如：

```c
int find(int[] arr) {
    int tmp = arr[0];
    for (int i = 1; i < arr.length; i++) {
        tmp = tmp ^ arr[i];
    }
    return tmp;
}
```

## 十进制转2/16进制

```c++
#include <iostream>
using namespace std;

int main() {
    int n;
    while (cin >> n) {
        cout << "decimal: " << n << endl;
        cout << "binary: ";
        for (int i = 2 * sizeof(int) - 1; i >= 0; i--) {
            int bit = ((n >> i) & 1);
            cout << bit;
        }
        cout << endl;
        cout << "hex: ";
        for (int i = 2 * sizeof(int) - 1; i >= 0; i--) { // sizeof(int)=4
            cout << "0123456789ABCDEF"[((n >> i * 4) & 0xF)]; // 0xF=(int)15=00001111, &0xF:取后四位
        }
        cout << endl;
    }
    return 0;
}
```

这个16进制的转换方法非常巧妙和简洁，可以经常翻看学习。
