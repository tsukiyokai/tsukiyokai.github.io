# 双链表 插入

双链表（DLL）有四种方式插入结点：

## 在表头插入

![dll_add_front](https://media.geeksforgeeks.org/wp-content/cdn-uploads/gq/2014/03/DLL_add_front1.png)

```c++
void push(Node** head_ref, int new_data) {
    Node* new_node = new Node();
    new_node->data = new_data;
    new_node->next = (*head_ref);
    new_node->prev = NULL;

    if ((*head_ref) != NULL) 
        (*head_ref)->prev = new_node;

    (*head_ref) = new_node;
}
```

## 在表尾插入

![dll_add_end](https://media.geeksforgeeks.org/wp-content/cdn-uploads/gq/2014/03/DLL_add_end1.png)

```c++
void append(Node** head_ref, int new_data)
{
    Node* new_node = new Node();
    Node* last = *head_ref;
    new_node->data = new_data;
    new_node->next = NULL;

    if (*head_ref == NULL) {
        new_node->prev = NULL;
        *head_ref = new_node;
        return;
    }

    while (last->next != NULL)
        last = last->next;

    last->next = new_node;
    new_node->prev = last;

    return;
}
```

## 在某点前插入

```c++
void insertBefore(Node** head_ref, Node* next_node, int new_data) /* 将给定的结点称为next_node */
{
    if (next_node == NULL) return;

    Node* new_node = new Node();
    new_node->data = new_data;

    new_node->prev = next_node->prev;

    new_node->next = next_node;
    if (new_node->prev != NULL) { /* 考虑给定结点左边是否指向NULL */
        next_node->prev = new_node;
        new_node->prev->next = new_node;
    }
    else *head_ref = new_node;
}
```

## 在某点后插入

![dll_add_middle](https://media.geeksforgeeks.org/wp-content/cdn-uploads/gq/2014/03/DLL_add_middle1.png)

```c++
void insertAfter(Node* prev_node, int new_data) { // 将给定的结点称为prev_node
    if (prev_node == NULL) return;

    Node* new_node = new Node();
    new_node->data = new_data;

    new_node->prev = prev_node;
    new_node->next = prev_node->next;

    if (new_node->next != NULL)
        new_node->next->prev = new_node;

    prev_node->next = new_node;
}
```

## 补充两条个人技巧

1. 因为双链表的插入操作需要按照正确的顺序断开或重新赋值某些指针，因此记忆这些操作的先后顺序可以提高一些效率。操作顺序为`4 1 2 3`。
2. 前插记得考虑左空的情况，后插记得考虑右空的情况，两端插入需要考虑是否为空表。
