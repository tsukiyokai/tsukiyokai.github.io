# 单链表 查找

## 查找符合条件的结点是否存在

编写一个函数，在给定的单链表中搜索给定的键值。如果在链表中，则该函数应返回true，否则返回false。

```c++
bool search(Node *head, int x)
```

### 1.迭代方法

```c++
bool search(Node* head, int x) {
    Node* current = head;
    while (current != NULL) {
        if (current->data == x) return true;
        current = current->next;
    }
    return false;
}
```

### 2.递归方法

```c++
bool search(Node* head, int x) {
    if (head == NULL) return false;
    if (head->data == x) return true;
    return search(head->next, x);
}
```

## 获取链表中第N个结点

编写一个GetNth()函数，该函数接受一个链表和一个整数索引，并返回存储在该索引位置的结点中的数据域。

```c++
int getNth(Node* head, int index) {
    Node* current = head;
    int count = 1;
    while (current) {
        if (count == index) return current->data;
        count++;
        current = current->next;
    }
    return -1;
}
```

另一种思路（递归）

```
getnth(node,n)
1. Initialize count = 1
2. if count==n
     return node->data
3. else
    return getnth(node->next,n-1)
```

具体实现

```c++
int GetNth(Node* head, int n) {
    if (head == NULL) return -1;
    int count = 1;
    if (count == n) return head->data;
    // recursively decrease n and increase head to next pointer
    return GetNth(head->next, n - 1);
}
```

这种方法并没有什么优势，逻辑也不是很直观，浅尝辄止了解即可。

## 获取链表中倒数第N个结点

例如倒数第三个结点是B。

![linkedlist](https://media.geeksforgeeks.org/wp-content/cdn-uploads/gq/2013/03/Linkedlist.png)

### 方法1 利用链表长度

主要思路

1. 计算链表的长度。让长度为len。

2. 从链表的开头出发，打印第`len – n + 1`个结点。

具体实现

```c++
int getNthR(Node* head, int n) { /* R = Reciprocal 倒数 */
    Node* temp = head;
    int len = getCount(head);
    if (len < n) return -1;
    temp = head;
    for (int i = 1; i < len - n + 1; i++)
        temp = temp->next;
    return temp->data;
}
```

### 方法2 使用两个指针

维护两个指针：main指针和reference指针。

初始化这两个指针使其指向head。首先，将reference指针从head开始出发移动n个结点。这一步使得main和ref拉开了n的距离，而“倒数第n”的意义正是目标结点与倒数第0的距离，倒数第1是尾结点，则倒数第0正是尾结点指针域所指向的null。所以接下来只需同时移动两个指针并保持此时的间距即可，移动的目的是：把ref从“正数第n”转变成“倒数第0”。换句话说，变动参考系。

然后，逐一移动两个指针，直到ref指针到达null为止。现在，main指针将恰好指向从右数起第n个结点。返回main指针即可。

下图是上述方法的模拟：

![img](https://media.geeksforgeeks.org/wp-content/cdn-uploads/20190624163533/Nth-node-from-end-of-the-Linked-List.png)

```c++
int getNthR(Node* head, int n) { /* R=Reciprocal 倒数 */
    Node* main_ptr = head;
    Node* ref_ptr = head;
    int diff = 0;

    if (head != NULL) {
        while (diff++ < n) {
            if (ref_ptr != NULL) ref_ptr = ref_ptr->next;
            else return -1;
        }

        while (ref_ptr != NULL) {
            main_ptr = main_ptr->next;
            ref_ptr = ref_ptr->next;
        }
        if (main_ptr != NULL)
            return main_ptr->data;
    }

    return -1;
}
```
