# 判断链表是否有环

![img](https://www.geeksforgeeks.org/wp-content/uploads/2009/04/Linked-List-Loop.gif)

## 解决方案1：散列方法

逐个遍历列表，并将结点地址放在哈希表中。在任何时候，如果到达NULL说明无环；如果当前结点的下一个地址指向哈希中先前存储的任何结点，则返回true。

```c++
bool detectLoop(struct Node* h) {
    set<Node*> s;
    while (h != NULL) {
        if (s.find(h) != s.end()) return true; // 当前结点不在set中
        s.insert(h);
        h = h->next;
    }
    return false;
}
```

## 解决方案2：修改链结点的数据结构，维护一个标志位

```c++
struct Node {
    int data;
    struct Node* next;
    int flag;
};

bool detectLoop(struct Node* h) {
    while (h != NULL) {
        if (h->flag == 1) return true;
        h->flag = 1;
        h = h->next;
    }
    return false;
}
```

## 解决方案3：Floyd判圈算法（这是最快的方法）

Floyd判圈算法(Floyd Cycle Detection Algorithm)，又称龟兔赛跑算法(Tortoise and Hare Algorithm)，是一个可以在有限状态机、迭代函数或者链表上判断是否存在环，求出该环的起点与长度的算法。该算法据高德纳称由美国科学家罗伯特·弗洛伊德发明，但这一算法并没有出现在罗伯特·弗洛伊德公开发表的著作中。

```
算法：
1. 使用两个指针遍历链表。
2. 满指针移动一个，快指针移动两个。
3. 如果两个指针在同一结点相遇，那么就有一个循环。如果指针不相交，那么链表就没有循环。
```
