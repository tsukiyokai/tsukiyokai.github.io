# 单链表 插入

## 正文

本文讨论的所有程序均基于下面的链结点表示形式。

```c++
class Node {
public:
    int data;
    Node* next;
};
```

本文讨论在链表中插入新结点的方法。可以通过三种方式添加结点：
1. 在链表的最前面
2. 在给定结点之后
3. 在链表的最末尾

## 在表首插入

新结点始终添加在给定链表的头结点之前，新近添加的结点将成为新的头结点。让我们将添加到链表最前面的函数称为push()，push()必须接收一个指向头指针的指针，因为push()必须使头结点指针指向新结点。(See [this](https://www.geeksforgeeks.org/how-to-write-functions-that-modify-the-head-pointer-of-a-linked-list/))

![linkedlist_insert_at_start](https://media.geeksforgeeks.org/wp-content/cdn-uploads/gq/2013/03/Linkedlist_insert_at_start.png)

```c++
/* 这里用到了二级指针 */
void push(Node** head_ref, int new_data) {
    Node* new_node = new Node(); /* 分配空间 */
    new_node->data = new_data; /* 向新结点的数据域存入数据 */
    new_node->next = (*head_ref); /* 让新结点的指针域指向原来的头结点 */
    (*head_ref) = new_node; /* 将头指针移动到新结点处 */
}
```

```tex
对上边代码片段的一些注解：
链表中第一个结点的存储位置叫做头指针，链表中可以没有头结点，但不能没有头指针。
根据初始化语句Node* head = NULL可知，head的类型是Node*，即结构体指针
head意义为指向头结点的指针/头指针，head内容为头结点地址，因此&head表示头结点地址的地址
new_node的类型是Node*，任何一个结点的类型都是Node*，任何一个结点的第一属性都是结点所在地址
箭头左边的next与右边的矩形边界看似分割，其实完全是一体。
head_ref的类型是Node**，即指向指针的指针，也就是指针的地址
传入参数是&head，即Node** head_ref = &head（本质为指针的指针 = 头结点地址的地址）。可推出：
head_ref=&head
*head_ref=head，即*head_ref=头指针
因此，new_node->next = *head_ref = head，即将新结点指向原头结点（第一个=是赋值，第二个=是等于）
进而，*head_ref = head = new_node，即将头指针更新为新结点处的地址
```

头指针中保存着头结点的地址，二级指针head_ref为指向指针的指针，head_ref解引用一次表示头指针，即头结点的地址，因此new_node->next = *head_ref表示将原头结点的地址赋值给新结点的指针域，即新结点指向头结点。

对二级指针的深入理解，参见： [深入理解二级指针.md](深入理解二级指针.md) 

## 在特定结点后插入

我们得到了某个结点的指针，新结点将插入到给定结点的后面。

![linkedlist_insert_middle](https://media.geeksforgeeks.org/wp-content/cdn-uploads/gq/2013/03/Linkedlist_insert_middle.png)

```c++
/* 在给定结点之后插入一个结点 */
void insertAfter(Node* prev_node, int new_data) {
    Node* new_node = new Node();
    new_node->data = new_data;
    new_node->next = prev_node->next;
    prev_node->next = new_node;
}
```

## 在表尾插入

由于链表通常由头指针表示，因此我们必须遍历该链表直至结束，然后将最后一个结点的next指针域更改为新结点的地址。

![linkedlist_insert_last](https://media.geeksforgeeks.org/wp-content/cdn-uploads/gq/2013/03/Linkedlist_insert_last.png)

```c++
// 在链表的最末尾添加一个结点
// 给定对列表头部的引用（指向指针的指针）和一个int，在链表末尾追加一个新结点
void append(Node** head_ref, int new_data) {
    Node* new_node = new Node();
    Node* last = *head_ref; /*last此时暂时指向头结点，稍后将遍历至尾结点*/
    new_node->data = new_data;
    new_node->next = NULL; /*待插入的新结点将成为新的末尾结点，因此将其next指针域置为NULL*/
    // 如果链表为空，则使新结点作为头结点
    if (*head_ref == NULL) /*为什么append要使用二级指针做形参？头指针不是没变吗？原因在这块，若链表为空时插入，头指针是会变的！*/
    {
        *head_ref = new_node;
        return;
    }
    // 如果链表非空，则遍历其直至last指向最后一个结点
    while (last->next != NULL)
        last = last->next;
    // 将遍历得到的尾结点指向新插入的结点，新结点成为新的尾巴
    last->next = new_node;
    return;
}
```

为什么第二个函数没有使用二级指针？

三个函数中，在头部新加结点一定会使头指针改变，在尾部添加结点也可能会使头指针改变（当链表本来为空），因此这两种情况都使用了二级指针。而在给定结点后面插入新结点则一定不会改变头指针，因此不需要使用二级指针。

```c++
void printList(Node* node) {
    while (node != NULL) {
        cout << " " << node->data;
        node = node->next;
    }
}

int main() {
    Node* head = NULL;
    append(&head, 6);
    push(&head, 7);
    push(&head, 1);
    append(&head, 4);
    insertAfter(head->next, 8);

    cout << "Created Linked list is:";
    printList(head);
    return 0;
}
```
