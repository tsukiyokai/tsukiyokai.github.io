不要将最长公共子串问题与最长公共子序列问题混淆！此问题与最长公共子序列问题的区别在于子序列不必是连续的，而子串却必须是。

最长公共子串问题是寻找一个最长的字符串，它同时是两个或多个已知字符串的子串。此问题可能有多种解决方案，其应用包括重复数据消除和剽窃检测。

摘自算法设计手册：当我们在多个文本中搜索相似性时，最长公共子序列的问题就会出现。一个特别重要的应用是在DNA序列中找到一致性。构建特定蛋白质的基因随着时间的推移而进化，但功能区必须保持一致才能正常工作。通过在不同物种中找到同一基因最长的共同子序列，我们了解了什么是随时间而保持不变的。

#### 示例

字符串“ABABC”、“BABCA”和“ABCBA”中最长的公共子字符串是长度为3的字符串“ABC”。其他不是最长的公共子串还有“A”、“AB”、“B”、“BA”、“BC”和“C”。

```tex
  ABABC
    |||
   BABCA
    |||
    ABCBA
```
#### 问题定义

给定两个字符串，分别为长度为m的串S和长度为n的串T，找到最长的字符串，它是S和T这两个字符串的子字符串。可推广到包含k个字符串的字符串集上成为k公共子串问题。

#### 算法

如果利用广义后缀树，我们可以在O(m+n)时间内求出S和T的最长公共子串长度和它们的起始位置（之所以说它们是因为LCS未必唯一）。

如果利用动态规划，则时间复杂度为O(mn)，而对推广的k公共子串问题，DP求解的时间复杂度为O(n1...nK)，而利用广义后缀树则需要O(NK)的时间复杂度（N为字符串集的总字符串长度，K为字符串的个数）。

最长公共子串问题是编辑距离的一个特例。

https://algorist.com/problems/Longest_Common_Substring.html

#### 动态规划

**伪代码**

```
function LCSubstr(S[1..r], T[1..n])
    L := array(1..r, 1..n)
    z := 0
    ret := {}

    for i := 1..r
        for j := 1..n
            if S[i] = T[j]
                if i = 1 or j = 1
                    L[i, j] := 1
                else
                    L[i, j] := L[i − 1, j − 1] + 1
                if L[i, j] > z
                    z := L[i, j]
                    ret := {S[i − z + 1..i]}
                else if L[i, j] = z
                    ret := ret ∪ {S[i − z + 1..i]}
            else
                L[i, j] := 0
    return ret
```

解释：数组L[i][j]用来存储分别以S[i]和T[j]结尾的前缀S[1...i]和T[1...j]的最长公共子序列的长度。变量z表示到目前为止找到的最长公共子串的长度。集合ret用来保存长度为z的子串。

为什么求最长公共子串的L表示的是最长公共子序列的长度啊？？？
lcs问题相当于给LCS问题加上了一个连续的限制。lcs的dp[][]根源于LCS的数组，但又有些改变。

为什么S[i]!=T[j]的时候L[i,j]:=0？
因为最长公共子串需要连续！！如果不等于的话，那么前面再多的相等都没用。这里的dp[i][j]更像是以i结尾且以j结尾的字符串最长的长度(粗略理解)。

**代码**

```c++
#include <bits/stdc++.h>
using namespace std;

int LCS(string X, string Y) {
    int m = X.length(), n = Y.length();
    vector<vector<int>> dp(m + 1, vector<int>(n + 1, 0));
    int res = 0;
    for (int i = 0; i <= m; i++) {
        for (int j = 0; j <= n; j++) {
            if (i == 0 || j == 0)
                dp[i][j] = 0;
            else if (X[i - 1] == Y[j - 1]) {
                dp[i][j] = dp[i - 1][j - 1] + 1;
                res = max(res, dp[i][j]);
            }
            else dp[i][j] = 0;
        }
    }
    return res;
}

int main() {
    string X = "OldSite:GeeksforGeeks.org";
    string Y = "NewSite:GeeksQuiz.com";

    cout << "最长公共子串长度为：" << LCS(X, Y); // 10
    return 0;
}
```

大概思路就是按自底向上顺序寻找最长公共后缀，有空还要去深入学一下后缀树。
