# 栈 介绍

![img](https://media.geeksforgeeks.org/wp-content/cdn-uploads/gq/2013/03/stack.png)

## 主要操作

Push()、Pop()、Peek()、isEmpty()

## 栈的应用

- [符号匹配](https://www.geeksforgeeks.org/check-for-balanced-parentheses-in-an-expression/)
- [中缀转后缀](http://quiz.geeksforgeeks.org/stack-set-2-infix-to-postfix/)或前缀
- 重做、撤销功能。在诸如editors, photoshop等许多场合。
- 前进、后退功能。在web浏览器中。
- 应用于许多算法，如[Tower of Hanoi](https://www.geeksforgeeks.org/recursive-functions/), [tree traversals](https://www.geeksforgeeks.org/618/), [stock span problem](https://www.geeksforgeeks.org/the-stock-span-problem/), [histogram problem](https://www.geeksforgeeks.org/largest-rectangular-area-in-a-histogram-set-1/).
- 回溯法。为了从当前状态恢复过来，我们需要存储先前的状态，为此我们需要堆栈。
- 图的算法。拓扑排序和强连通分量。
- 内存管理。在内存管理中，任何现代计算机都将堆栈用作运行目的的主要管理方法。计算机系统中运行的每个程序都有自己的内存分配。
- 字符串反转。

## 实现方式

### 用数组

容易实现。但是非动态。

```c++
#include <iostream>

using namespace std;

#define MAX 1000

class Stack {
    int top;
public:
    int a[MAX] = {};
    Stack() { top = -1; }
    bool push(int x);
    int pop();
    int peek();
    bool isEmpty();
};

bool Stack::push(int x) {
    if (top >= (MAX - 1)) return false;
    else {
        a[++top] = x;
        cout << x << " pushed into stack\n";
        return true;
    }
}

int Stack::pop() {
    if (top < 0) return -1;
    else {
        int x = a[top--];
        return x;
    }
}

int Stack::peek() {
    if (top < 0) return -1;
    else {
        int x = a[top];
        return x;
    }
}

bool Stack::isEmpty() {
    return (top < 0);
}

int main() {
    Stack s;
    s.push(10);
    s.push(20);
    s.push(30);
    cout << s.pop() << " Popped from stack\n";
    return 0;
}
```

### 用链表

动态大小。但是需要额外内存开销。

```c++
#include <bits/stdc++.h>
using namespace std;

class StackNode {
public:
    int data;
    StackNode* next;
};

StackNode* newNode(int data) {
    StackNode* stackNode = new StackNode();
    stackNode->data = data;
    stackNode->next = NULL;
    return stackNode;
}

int isEmpty(StackNode* top) {
    return top == NULL;
}

void push(StackNode** top, int data) {
    StackNode* stackNode = newNode(data);
    stackNode->next = *top;
    *top = stackNode;
    cout << data << " pushed to stack.\n";
}

/* 这是自己实现的，其实标准库中自带的是void pop()，不是int pop()，因此不能顺便借用pop的返回值 */
int pop(StackNode** top) {
    if (isEmpty(*top)) return INT_MIN;
    StackNode* temp = *top;
    *top = (*top)->next;
    int popped = temp->data;
    free(temp);
    return popped;
}

int peek(StackNode* top) {
    if (isEmpty(top)) return INT_MIN;
    return top->data;
}

int main() {
    StackNode* top = NULL;
    push(&top, 10);
    push(&top, 20);
    push(&top, 30);
    cout << pop(&top) << " popped from stack.\n";
    cout << "Top element is " << peek(top) << endl;
    cout << (isEmpty(top) ? "Empty." : "NonEmpty.") << endl;
    return 0;
}
```
