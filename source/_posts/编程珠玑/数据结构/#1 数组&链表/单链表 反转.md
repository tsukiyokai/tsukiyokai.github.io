# 单链表 反转

```
Input: Head of following linked list 
1->2->3->4->5->NULL 
Output: Linked list should be changed to, 
5->4->3->2->1->NULL
```

## 迭代方法

### 算法

```
1. Initialize three pointers: a) prev as NULL b) curr as head c) next as NULL.
2. Iterate through the linked list. In loop, do following.
// Before changing next of current, store next node 
    next = curr->next
// Now change next of current. This is where actual reversing happens 
    curr->next = prev 
// Move prev and curr one step forward 
    prev = curr 
    curr = next
```

![Lightbox](https://media.geeksforgeeks.org/wp-content/cdn-uploads/RGIF2.gif)

### 代码

```c++
void reverse() {
    Node* current = head;
    Node* prev = NULL, * next = NULL;
    while (current != NULL) {
        next = current->next;
        current->next = prev;
        prev = current;
        current = next;
    }
    head = prev;
}
```

### 分析

```
Time Complexity: O(n) 
Space Complexity: O(1)
```

## 递归方法

### 算法

```tex
1) Divide the list in two parts:
    first node and rest of the linked list.
2) Call reverse for the rest of the linked list.
3) Link rest to first.
4) Fix head pointer
```

```
          head(0)  ->  1  ->  2  ->  3  ->  null
          head(0)  ->  1  <-  2  <-  3(rest)null
          head(0)  <-  1  <-  2  <-  3      null
null  <-  head(0)  <-  1  <-  2  <-  3      null
```

### 代码

```c++
Node* reverse(Node* head) {
    if (head == NULL || head->next == NULL) return head;
    Node* rest = reverse(head->next);
    head->next->next = head;
    head->next = NULL;
    return rest;
}
```

## 递归之禅

Trust that the recursive call on each sub-tree works and concentrate on assembling the outputs of the recursive calls to build the result. It's too complex to delve into how each recursive call is going to work -- trust that it did work and assemble the answer from there.
