# 双链表 反转

![img](https://media.geeksforgeeks.org/wp-content/uploads/reverse_lincked-list1.jpg)

![img](https://media.geeksforgeeks.org/wp-content/uploads/reverse_lincked-list2.jpg)

## 方法1 简单方法

对于每一个结点，交换其两个指针域的内容。最后将头指针指向尾结点。

```c++
void reverse(Node** head_ref) {
    Node* temp = NULL;
    Node* current = *head_ref;

    while (current != NULL) { /* swap */
        temp = current->prev;
        current->prev = current->next;
        current->next = temp;

        current = current->prev;
    }

    if (temp != NULL) *head_ref = temp->prev;
}
```

## 方法2 使用堆栈

将结点的数据域连续压入栈中，然后不断弹出元素并更新双链表。
