# 双端队列 双链表实现

头————尾（allows insert and delete at both ends）

## 主要功能

```
insertFront() : Adds an item at the front of Deque.
insertRear()  : Adds an item at the rear of Deque.
deleteFront() : Deletes an item from front of Deque.
deleteRear()  : Deletes an item from rear of Deque.

getFront() : Gets the front item from queue.
getRear()  : Gets the last item from queue.
isEmpty()  : Checks whether Deque is empty or not.
size()     : Gets number of elements in Deque.
erase()    : Deletes all the elements from Deque.
```

## 伪代码

**Insertion at Front end**

```
Allocate space for a newNode of doubly linked list.
IF newNode == NULL, then // 内存不够分配新结点
    print "Overflow"
ELSE
    IF front == NULL, then // 空队列
        rear = front = newNode
    ELSE // 非空队列
      newNode->next = front // 前后结点桥接
      front->prev = newNode
      front = newNode // 改变端点指针指向
```

**Insertion at Rear end**

```
Allocate space for a newNode of doubly linked list.
IF newNode == NULL, then
    print "Overflow"
ELSE
    IF rear == NULL, then
        front = rear = newNode
    ELSE
        newNode->prev = rear
        rear->next = newNode
        rear = newNode
```

**Deletion from Front end**

```
IF front == NULL
    print "Underflow"
ELSE
    Initialize temp = front
    front = front->next
    IF front == NULL
        rear = NULL
    ELSE
        front->prev = NULL // 容易遗漏这句，头结点前面没有结点，但头结点的prev指针域还是要记得置为NULL
    Deallocate space for temp
```

**Deletion from Rear end**

```
IF front == NULL
    print "Underflow"
ELSE
    Initialize temp = rear
    rear = rear->prev
    IF rear == NULL
        front = NULL
    ELSE
        rear->next = NULL
    Deallocate space for temp
```
