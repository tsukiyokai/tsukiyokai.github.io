本文所有程序均基于如下链结点表示形式。

```c++
class Node {
public:
    int data;
    Node* next;
};
```

本文讨论在链表中插入新结点的方法，可以通过三种方式添加结点：

1. 在链表的最前面
2. 在给定结点之后
3. 在链表的最末尾

## 在表首插入

新结点始终添加在给定链表的头结点之前，新近添加的结点将成为新的头结点。让我们将添加到链表最前面的函数称为 push()，push()必须接收一个指向头指针的指针，因为 push()必须使头结点指针指向新结点。(See [this](https://www.geeksforgeeks.org/how-to-write-functions-that-modify-the-head-pointer-of-a-linked-list/))

![linkedlist_insert_at_start](https://media.geeksforgeeks.org/wp-content/cdn-uploads/gq/2013/03/Linkedlist_insert_at_start.png)

```c++
void push(Node** head_ref, int new_data) {
    Node* new_node = new Node();
    new_node->data = new_data;
    new_node->next = (*head_ref);
    (*head_ref) = new_node;
}
```

头指针中保存着头结点的地址，二级指针 head_ref 为指向指针的指针，head_ref 解引用一次表示头指针，即头结点的地址，因此`new_node->next = *head_ref`表示将原头结点的地址赋值给新结点的指针域，即新结点指向头结点。

## 在特定结点后插入

我们得到了某个结点的指针，新结点将插入到给定结点的后面。

![linkedlist_insert_middle](https://media.geeksforgeeks.org/wp-content/cdn-uploads/gq/2013/03/Linkedlist_insert_middle.png)

```c++
void insertAfter(Node* prev_node, int new_data) {
    Node* new_node = new Node();
    new_node->data = new_data;
    new_node->next = prev_node->next;
    prev_node->next = new_node;
}
```

## 在表尾插入

由于链表通常由头指针表示，因此我们必须遍历该链表直至结束，然后将最后一个结点的 next 指针域更改为新结点的地址。

![linkedlist_insert_last](https://media.geeksforgeeks.org/wp-content/cdn-uploads/gq/2013/03/Linkedlist_insert_last.png)

```c++
void append(Node** head_ref, int new_data) {
    Node* new_node = new Node();
    Node* last = *head_ref;
    new_node->data = new_data;
    new_node->next = NULL;
    if (*head_ref == NULL) {
        *head_ref = new_node;
        return;
    }
    while (last->next != NULL)
        last = last->next;
    last->next = new_node;
    return;
}
```

为什么第二个函数没有使用二级指针？三个函数中，在头部新加结点一定会使头指针改变，在尾部添加结点也可能会使头指针改变（当链表本来为空），因此这两种情况都使用了二级指针。而在给定结点后面插入新结点则一定不会改变头指针，因此不需要使用二级指针。

```c++
void printList(Node* node) {
    while (node != NULL) {
        cout << " " << node->data;
        node = node->next;
    }
}

int main() {
    Node* head = NULL;
    append(&head, 6);
    push(&head, 7);
    push(&head, 1);
    append(&head, 4);
    insertAfter(head->next, 8);

    cout << "Created Linked list is:";
    printList(head);
    return 0;
}
```
