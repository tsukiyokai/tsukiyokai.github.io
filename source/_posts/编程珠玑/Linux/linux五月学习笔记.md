> 刚开始linux复建，记点东西。

#### 进入界面

进入ubt子系统的图形界面

```shell
$ xfce4-session
```

#### Super User DO

获取包的更新信息并安装更新

```shell
$ sudo apt-get update && sudo apt-get upgrade
```

#### man

ascii表

```shell
$ man ascii
```

#### cal|calendar|date

历史上的今天

```shell
$ calendar -t 1997.04.09
```

万年历

```shell
$ cal 1997
```
