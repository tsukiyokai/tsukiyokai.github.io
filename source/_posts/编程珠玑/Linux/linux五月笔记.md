> 刚开始linux复建，记点东西。

进入ubt子系统的图形界面

```shell
$ xfce4-session
```

获取系统中包的更新信息并进行更新

```shell
$ sudo apt-get update && sudo apt-get upgrade
```

查看ascii详细信息

```shell
$ man ascii
```

历史上的今天

```shell
$ calendar -t 1997.04.09
```

万年历

```shell
$ cal 1997
```
