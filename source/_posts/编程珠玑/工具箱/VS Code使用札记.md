打开VS Code
->Ctrl+H打开替换
->选中使用正则表达式
->查找栏输入正则表达式，替换栏置空
->Ctrl+Alt+Enter全部替换

#### 批量删除注释

```tex
行注释
//[\s\S]*?\n

块注释
/\*(.|\r\n|\n)*?\*/

所有注释
\/\*[\s\S]*\*\/|\/\/.*

html注释
<!--([\s\S|\r]*?)-->
```

#### 每行首尾添加字符串

```tex
上尖号代表每行的开头
^

美元符号代表每行的结尾
$
```

#### 选中含某个串的所有行

```tex
^(.*)abbrlink(.*)\n
```
