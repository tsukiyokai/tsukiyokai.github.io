堆简介

发表于 2021-04-16 分类于 编程珠玑 → 数据结构与算法分析 阅读次数：

堆 heap

堆的概念

n个元素序列 { k1, k2, k3, k4, k5, k6, …, kn } 当且仅当满足以下关系时才会被称为堆：
ki <= k2i, ki <= k2i+1 或 ki >= k2i, ki >= k2i+1 (i = 1, 2, 3, 4, …, n/2)
如果数组的下标是从0开始，那么需要满足：
ki <= k2i+1, ki <= k2i+2 或者 ki >= k2i+1, ki >= k2i+2 (i = 0, 1, 2, 3, …, n/2)

比如 { 1, 3, 5, 10, 15, 9 } 这个序列就满足 [1 <= 3; 1 <= 5], [3 <= 10; 3 <= 15], [5 <= 9] 这3个条件，这个序列就是一个堆。

所以堆其实是一个序列或数组，如果这个序列满足上述条件，那么就把这个序列看成堆。另外，堆又被称作堆积树。

堆的定义

定义：一棵完全二叉树，其每个节点都有一个值。特点是根节点值最小（或最大），且根节点的两个子树也是一个堆。根节点值最小的堆称为小根堆，根节点值最大的堆称为大根堆。
学科：计算机科学技术 理论计算机科学 数据结构
来源：计算机科学技术名词第三版

二叉堆 binary heap

堆的实现通常是通过构造二叉堆，因为二叉堆应用很普遍，当不加限定时，堆通常指的就是二叉堆。

二叉堆的数组表示

二叉堆是一棵完全二叉树。二叉堆通常表示为数组的形式。表示方法如下：

  1. 根节点位于arr[0]
  2. 第i个节点的相关节点可以表示为：
    父节点：arr[(i-1)/2] 向下取整
    左孩子：arr[2*i+1]
    右孩子：arr[2*i+2]

把堆用数组表示所使用的方法是进行层序遍历。

二叉堆总是满足以下两种堆序性（heap order）之一：
最大堆：父节点的键值总是大于等于任何一个子节点的键值。
最小堆：父节点的键值总是小于等于任何一个子节点的键值。

二叉堆的应用

  1. 堆排序。堆排序使用二叉堆在O(nlogn)时间内对数组进行排序。
  2. 优先级队列。使用二叉堆可以有效地实现优先级队列，因为它支持在O(logn)时间内执行insert()、delete()和extractmax()、decreaseKey()等操作。二项式堆和斐波那契堆是二叉堆的变体。用堆实现的优先级队列主要用于Prim算法和Dijkstra算法等图算法中。
  3. 图的算法。优先级队列特别适用于Dijkstra最短路径和Prim最小生成树等图算法。
  4. 顺序统计量（Order Statistics）。使用堆可以有效地解决许多问题，如：
    a）数组中第K大的元素。
    b）对基本有序的数组进排序。
    c）合并K个有序数组。

二叉堆的基本操作

操作 描述 时间复杂度
build 创建一个空堆 O(n)*
insert 向堆中插入一个新元素 O(logn)
update 提升新元素使其符合堆的性质 O(logn)
get 获取当前堆顶元素即根节点的值 O(1)
delete 删除堆顶元素 O(logn)
heapify 使删除堆顶元素的堆再次成为堆 O(logn)
union 将两个堆合并成一个有效堆 O(n), O(klogn)**

注：
*构造堆的时间复杂度为O(n)的相关证明参见：How can building a heap be O(n) time complexity?
**k是两个堆之间较小那个堆的长度

某些堆实现还支持其他的一些操作，如斐波那契堆支持检查一个堆中是否存在某个元素。

为了简洁起见，下面以最小堆的相关操作为例进行说明：

  1. getMini()：返回最小堆的根节点。时间复杂度为O(1)。
  2. insert()：插入新节点需要O(logn)时间。我们在完全二叉树最下层的末尾添加一个新键。如果新键大于其父键，则不需要执行任何操作。否则，需要遍历来修复冲突的heap属性。
  3. delete()：删除节点也需要O(logn)时间。因为删除操作是通过调用decreaseKey()将待删除的节点替换为负无穷大（注意是负无穷大而不是无穷小）。负无穷大的值必须到达最小堆的根节点处，此时我们再调用extractMin()来移除该节点。
  4. extractMin()：从最小堆中删除根节点。时间复杂度为O(logn)，因为删除后还必须通过调用heapify()来维护heap属性。
  5. decreaseKey()：减少某个关键字到指定的值。时间复杂度为O(logn)。一个节点减少后必然依旧小于子节点；如果一个节点减少后的值仍然大于其父节点，那么我们不需要做任何事情。否则，需要遍历来修复冲突的heap属性。
