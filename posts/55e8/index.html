е Ҷзҡ„жҰӮеҝө

nдёӘе…ғзҙ еәҸеҲ— { k1, k2, k3, k4, k5, k6, вҖҰ, kn } еҪ“дё”д»…еҪ“ж»Ўи¶ід»ҘдёӢе…ізі»ж—¶жүҚдјҡиў«з§°дёәе Ҷпјҡ
ki <= k2i, ki <= k2i+1 жҲ– ki >= k2i, ki >= k2i+1 (i = 1, 2, 3, 4, вҖҰ, n/2)
еҰӮжһңж•°з»„зҡ„дёӢж ҮжҳҜд»Һ0ејҖе§ӢпјҢйӮЈд№ҲйңҖиҰҒж»Ўи¶іпјҡ
ki <= k2i+1, ki <= k2i+2 жҲ–иҖ… ki >= k2i+1, ki >= k2i+2 (i = 0, 1, 2, 3, вҖҰ, n/2)

жҜ”еҰӮ { 1, 3, 5, 10, 15, 9 } иҝҷдёӘеәҸеҲ—е°ұж»Ўи¶і [1 <= 3; 1 <= 5], [3 <= 10; 3 <= 15], [5 <= 9] иҝҷ3дёӘжқЎд»¶пјҢиҝҷдёӘеәҸеҲ—е°ұжҳҜдёҖдёӘе ҶгҖӮ

жүҖд»Ҙе Ҷе…¶е®һжҳҜдёҖдёӘеәҸеҲ—жҲ–ж•°з»„пјҢеҰӮжһңиҝҷдёӘеәҸеҲ—ж»Ўи¶ідёҠиҝ°жқЎд»¶пјҢйӮЈд№Ҳе°ұжҠҠиҝҷдёӘеәҸеҲ—зңӢжҲҗе ҶгҖӮеҸҰеӨ–пјҢе ҶеҸҲиў«з§°дҪңе Ҷз§Ҝж ‘гҖӮ

е Ҷзҡ„е®ҡд№ү

е®ҡд№үпјҡдёҖжЈөе®Ңе…ЁдәҢеҸүж ‘пјҢе…¶жҜҸдёӘиҠӮзӮ№йғҪжңүдёҖдёӘеҖјгҖӮзү№зӮ№жҳҜж №иҠӮзӮ№еҖјжңҖе°ҸпјҲжҲ–жңҖеӨ§пјүпјҢдё”ж №иҠӮзӮ№зҡ„дёӨдёӘеӯҗж ‘д№ҹжҳҜдёҖдёӘе ҶгҖӮж №иҠӮзӮ№еҖјжңҖе°Ҹзҡ„е Ҷз§°дёәе°Ҹж №е ҶпјҢж №иҠӮзӮ№еҖјжңҖеӨ§зҡ„е Ҷз§°дёәеӨ§ж №е ҶгҖӮ
еӯҰз§‘пјҡи®Ўз®—жңәз§‘еӯҰжҠҖжңҜ зҗҶи®әи®Ўз®—жңәз§‘еӯҰ ж•°жҚ®з»“жһ„
жқҘжәҗпјҡи®Ўз®—жңәз§‘еӯҰжҠҖжңҜеҗҚиҜҚз¬¬дёүзүҲ

дәҢеҸүе Ҷ binary heap

е Ҷзҡ„е®һзҺ°йҖҡеёёжҳҜйҖҡиҝҮжһ„йҖ дәҢеҸүе ҶпјҢеӣ дёәдәҢеҸүе Ҷеә”з”ЁеҫҲжҷ®йҒҚпјҢеҪ“дёҚеҠ йҷҗе®ҡж—¶пјҢе ҶйҖҡеёёжҢҮзҡ„е°ұжҳҜдәҢеҸүе ҶгҖӮ

дәҢеҸүе Ҷзҡ„ж•°з»„иЎЁзӨә

дәҢеҸүе ҶжҳҜдёҖжЈөе®Ңе…ЁдәҢеҸүж ‘гҖӮдәҢеҸүе ҶйҖҡеёёиЎЁзӨәдёәж•°з»„зҡ„еҪўејҸгҖӮиЎЁзӨәж–№жі•еҰӮдёӢпјҡ

  1. ж №иҠӮзӮ№дҪҚдәҺarr[0]
  2. з¬¬iдёӘиҠӮзӮ№зҡ„зӣёе…іиҠӮзӮ№еҸҜд»ҘиЎЁзӨәдёәпјҡ
    зҲ¶иҠӮзӮ№пјҡarr[(i-1)/2] еҗ‘дёӢеҸ–ж•ҙ
    е·Ұеӯ©еӯҗпјҡarr[2*i+1]
    еҸіеӯ©еӯҗпјҡarr[2*i+2]

жҠҠе Ҷз”Ёж•°з»„иЎЁзӨәжүҖдҪҝз”Ёзҡ„ж–№жі•жҳҜиҝӣиЎҢеұӮеәҸйҒҚеҺҶгҖӮ

дәҢеҸүе ҶжҖ»жҳҜж»Ўи¶ід»ҘдёӢдёӨз§Қе ҶеәҸжҖ§пјҲheap orderпјүд№ӢдёҖпјҡ
жңҖеӨ§е ҶпјҡзҲ¶иҠӮзӮ№зҡ„й”®еҖјжҖ»жҳҜеӨ§дәҺзӯүдәҺд»»дҪ•дёҖдёӘеӯҗиҠӮзӮ№зҡ„й”®еҖјгҖӮ
жңҖе°Ҹе ҶпјҡзҲ¶иҠӮзӮ№зҡ„й”®еҖјжҖ»жҳҜе°ҸдәҺзӯүдәҺд»»дҪ•дёҖдёӘеӯҗиҠӮзӮ№зҡ„й”®еҖјгҖӮ

дәҢеҸүе Ҷзҡ„еә”з”Ё

  1. е ҶжҺ’еәҸгҖӮе ҶжҺ’еәҸдҪҝз”ЁдәҢеҸүе ҶеңЁO(nlogn)ж—¶й—ҙеҶ…еҜ№ж•°з»„иҝӣиЎҢжҺ’еәҸгҖӮ
  2. дјҳе…Ҳзә§йҳҹеҲ—гҖӮдҪҝз”ЁдәҢеҸүе ҶеҸҜд»Ҙжңүж•Ҳең°е®һзҺ°дјҳе…Ҳзә§йҳҹеҲ—пјҢеӣ дёәе®ғж”ҜжҢҒеңЁO(logn)ж—¶й—ҙеҶ…жү§иЎҢinsert()гҖҒdelete()е’Ңextractmax()гҖҒdecreaseKey()зӯүж“ҚдҪңгҖӮдәҢйЎ№ејҸе Ҷе’Ңж–җжіўйӮЈеҘ‘е ҶжҳҜдәҢеҸүе Ҷзҡ„еҸҳдҪ“гҖӮз”Ёе Ҷе®һзҺ°зҡ„дјҳе…Ҳзә§йҳҹеҲ—дё»иҰҒз”ЁдәҺPrimз®—жі•е’ҢDijkstraз®—жі•зӯүеӣҫз®—жі•дёӯгҖӮ
  3. еӣҫзҡ„з®—жі•гҖӮдјҳе…Ҳзә§йҳҹеҲ—зү№еҲ«йҖӮз”ЁдәҺDijkstraжңҖзҹӯи·Ҝеҫ„е’ҢPrimжңҖе°Ҹз”ҹжҲҗж ‘зӯүеӣҫз®—жі•гҖӮ
  4. йЎәеәҸз»ҹи®ЎйҮҸпјҲOrder StatisticsпјүгҖӮдҪҝз”Ёе ҶеҸҜд»Ҙжңүж•Ҳең°и§ЈеҶіи®ёеӨҡй—®йўҳпјҢеҰӮпјҡ
    aпјүж•°з»„дёӯз¬¬KеӨ§зҡ„е…ғзҙ гҖӮ
    bпјүеҜ№еҹәжң¬жңүеәҸзҡ„ж•°з»„иҝӣжҺ’еәҸгҖӮ
    cпјүеҗҲе№¶KдёӘжңүеәҸж•°з»„гҖӮ

дәҢеҸүе Ҷзҡ„еҹәжң¬ж“ҚдҪң

ж“ҚдҪң жҸҸиҝ° ж—¶й—ҙеӨҚжқӮеәҰ
build еҲӣе»әдёҖдёӘз©әе Ҷ O(n)*
insert еҗ‘е ҶдёӯжҸ’е…ҘдёҖдёӘж–°е…ғзҙ  O(logn)
update жҸҗеҚҮж–°е…ғзҙ дҪҝе…¶з¬ҰеҗҲе Ҷзҡ„жҖ§иҙЁ O(logn)
get иҺ·еҸ–еҪ“еүҚе ҶйЎ¶е…ғзҙ еҚіж №иҠӮзӮ№зҡ„еҖј O(1)
delete еҲ йҷӨе ҶйЎ¶е…ғзҙ  O(logn)
heapify дҪҝеҲ йҷӨе ҶйЎ¶е…ғзҙ зҡ„е ҶеҶҚж¬ЎжҲҗдёәе Ҷ O(logn)
union е°ҶдёӨдёӘе ҶеҗҲе№¶жҲҗдёҖдёӘжңүж•Ҳе Ҷ O(n), O(klogn)**

жіЁпјҡ
*жһ„йҖ е Ҷзҡ„ж—¶й—ҙеӨҚжқӮеәҰдёәO(n)зҡ„зӣёе…іиҜҒжҳҺеҸӮи§ҒпјҡHow can building a heap be O(n) time complexity?
**kжҳҜдёӨдёӘе Ҷд№Ӣй—ҙиҫғе°ҸйӮЈдёӘе Ҷзҡ„й•ҝеәҰ

жҹҗдәӣе Ҷе®һзҺ°иҝҳж”ҜжҢҒе…¶д»–зҡ„дёҖдәӣж“ҚдҪңпјҢеҰӮж–җжіўйӮЈеҘ‘е Ҷж”ҜжҢҒжЈҖжҹҘдёҖдёӘе ҶдёӯжҳҜеҗҰеӯҳеңЁжҹҗдёӘе…ғзҙ гҖӮ

дёәдәҶз®ҖжҙҒиө·и§ҒпјҢдёӢйқўд»ҘжңҖе°Ҹе Ҷзҡ„зӣёе…іж“ҚдҪңдёәдҫӢиҝӣиЎҢиҜҙжҳҺпјҡ

  1. getMini()пјҡиҝ”еӣһжңҖе°Ҹе Ҷзҡ„ж №иҠӮзӮ№гҖӮж—¶й—ҙеӨҚжқӮеәҰдёәO(1)гҖӮ
  2. insert()пјҡжҸ’е…Ҙж–°иҠӮзӮ№йңҖиҰҒO(logn)ж—¶й—ҙгҖӮжҲ‘д»¬еңЁе®Ңе…ЁдәҢеҸүж ‘жңҖдёӢеұӮзҡ„жң«е°ҫж·»еҠ дёҖдёӘж–°й”®гҖӮеҰӮжһңж–°й”®еӨ§дәҺе…¶зҲ¶й”®пјҢеҲҷдёҚйңҖиҰҒжү§иЎҢд»»дҪ•ж“ҚдҪңгҖӮеҗҰеҲҷпјҢйңҖиҰҒйҒҚеҺҶжқҘдҝ®еӨҚеҶІзӘҒзҡ„heapеұһжҖ§гҖӮ
  3. delete()пјҡеҲ йҷӨиҠӮзӮ№д№ҹйңҖиҰҒO(logn)ж—¶й—ҙгҖӮеӣ дёәеҲ йҷӨж“ҚдҪңжҳҜйҖҡиҝҮи°ғз”ЁdecreaseKey()е°Ҷеҫ…еҲ йҷӨзҡ„иҠӮзӮ№жӣҝжҚўдёәиҙҹж— з©·еӨ§пјҲжіЁж„ҸжҳҜиҙҹж— з©·еӨ§иҖҢдёҚжҳҜж— з©·е°ҸпјүгҖӮиҙҹж— з©·еӨ§зҡ„еҖјеҝ…йЎ»еҲ°иҫҫжңҖе°Ҹе Ҷзҡ„ж №иҠӮзӮ№еӨ„пјҢжӯӨж—¶жҲ‘д»¬еҶҚи°ғз”ЁextractMin()жқҘз§»йҷӨиҜҘиҠӮзӮ№гҖӮ
  4. extractMin()пјҡд»ҺжңҖе°Ҹе ҶдёӯеҲ йҷӨж №иҠӮзӮ№гҖӮж—¶й—ҙеӨҚжқӮеәҰдёәO(logn)пјҢеӣ дёәеҲ йҷӨеҗҺиҝҳеҝ…йЎ»йҖҡиҝҮи°ғз”Ёheapify()жқҘз»ҙжҠӨheapеұһжҖ§гҖӮ
  5. decreaseKey()пјҡеҮҸе°‘жҹҗдёӘе…ій”®еӯ—еҲ°жҢҮе®ҡзҡ„еҖјгҖӮж—¶й—ҙеӨҚжқӮеәҰдёәO(logn)гҖӮдёҖдёӘиҠӮзӮ№еҮҸе°‘еҗҺеҝ…з„¶дҫқж—§е°ҸдәҺеӯҗиҠӮзӮ№пјӣеҰӮжһңдёҖдёӘиҠӮзӮ№еҮҸе°‘еҗҺзҡ„еҖјд»Қз„¶еӨ§дәҺе…¶зҲ¶иҠӮзӮ№пјҢйӮЈд№ҲжҲ‘д»¬дёҚйңҖиҰҒеҒҡд»»дҪ•дәӢжғ…гҖӮеҗҰеҲҷпјҢйңҖиҰҒйҒҚеҺҶжқҘдҝ®еӨҚеҶІзӘҒзҡ„heapеұһжҖ§гҖӮ
